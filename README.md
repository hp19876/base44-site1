# Base44 Site 1 - IT Equipment E-Commerce Store

A modern, conversion-optimized e-commerce website built with Base44 for selling IT equipment including UPS, VoIP phones, Ubiquiti network gears, Netgear switches, Mikrotik, QNAP, Synology, Yealink, Atdec, and server solutions.

## Project Structure

### Folders
- **entities/** - Data models for Product and Order entities
- - **src/api/** - API integration and backend services
  - - **src/components/** - Reusable React components
    - - **src/hooks/** - Custom React hooks
      - - **src/lib/** - Utility functions and libraries
        - - **src/pages/** - Page components for different routes
          - - **src/utils/** - General utility functions
           
            - ### Key Files
            - - **App.jsx** - Main application component
              - - **main.jsx** - Application entry point
                - - **index.html** - HTML template
                  - - **index.css** - Global styles
                    - - **Layout.jsx** - Layout wrapper component
                      - - **pages.config.js** - Page configuration
                       
                        - ### Configuration Files
                        - - **vite.config.js** - Vite build configuration
                          - - **tailwind.config.js** - Tailwind CSS configuration
                            - - **postcss.config.js** - PostCSS configuration
                              - - **eslint.config.js** - ESLint rules
                                - - **jsconfig.json** - JavaScript configuration
                                  - - **components.json** - Component registry
                                    - - **package.json** - Project dependencies
                                     
                                      - ## Features
                                     
                                      - - Product listing with filtering and search
                                        - - Detailed product pages with images and specifications
                                          - - Shopping cart functionality
                                            - - Multi-step checkout process
                                              - - Responsive design for all devices
                                                - - Trust elements and security badges
                                                  - - Order management system

                                                    - 1. Install dependencies: `npm install`
                                                      2. 2. Start development server: `npm run dev`
                                                         3. 3. Build for production: `npm run build`
